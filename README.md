# MarkDownEditor & CSharpMobileApp

Репозиторий содержит два независимых проекта на .NET:

1. **MarkdownEditor** — десктопное приложение на Avalonia для редактирования Markdown с предпросмотром.
2. **CSharpMobileApp** — мобильное приложение на .NET MAUI (Android/Windows) с ToDo-листом.

Оба проекта ориентированы на учебные цели (курс по мобильной разработке / C#), но имеют рабочую функциональность и могут использоваться как базовые шаблоны.

---

## Общие требования

- .NET SDK **8.0**
- Visual Studio 2022 с установленными workloads:
  - **.NET Desktop Development**
  - **.NET Multi-platform App UI development (MAUI)** — для `CSharpMobileApp`
- Git (если клонируешь репозиторий)
